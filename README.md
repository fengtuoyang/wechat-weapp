# wechat-weapp-doubanfilm
微信小程序--豆瓣电影--音乐播放--地图

豆瓣电影api文档好像关闭了，但是接口还可以用。

    正在热映：http://t.yushu.im/v2/movie/in_theaters
    即将上映：http://t.yushu.im/v2/movie/coming_soon
    Top250：http://t.yushu.im/v2/movie/top250
    参数start=0&count=3
